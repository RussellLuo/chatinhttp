ChatInHTTP
==========

Chat in HTTP between client and server.

Usage
-----

###1 Chat with local server

1) start local server

    python server.py

2) chat with it

    python ChatInHTTP.py localhost 8000

###2 Chat with remote server

    python ChatInHTTP.py www.python.org

Tips
----

CTRL-D    EOF of stdin
CTRL-C    exit
